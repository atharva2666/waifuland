# **App Name**: WaifuVault

## Core Features:

- Waifu Image Gallery: Display a random anime waifu image (including NSFW options) fetched from the waifu.Pics API upon loading or refresh.
- Category Selector: Allow users to select from different waifu categories (e.g., 'waifu', 'neko', 'shinobu', 'nsfw') to filter image display.
- Generate New Waifu: A dedicated button to fetch and display a new random image within the currently selected category (including NSFW options).
- Image Details Display: When an image is clicked, display detailed information about the anime it belongs to, including its launch day, by querying external databases (e.g., MyAnimeList, AniList).
- Image Download: Provide a simple button to allow users to download the currently displayed waifu image.

## Style Guidelines:

- Primary color: A sophisticated deep lavender (#6E52AD) to evoke a modern, dreamy digital aesthetic.
- Background color: A very light, subtle lavender-grey (#F6F3F9) providing a clean, ethereal canvas for the artwork.
- Accent color: A vibrant, clear sky blue (#598BF2) to highlight interactive elements and add a pop of digital energy.
- Headlines and body text font: 'Poppins', a geometric sans-serif, chosen for its contemporary precision and clean lines, perfectly complementing digital art.
- Use minimalist line-art icons that complement the clean interface, focusing on actions like download, category change, regeneration, and viewing details.
- A responsive, centered layout prioritizing the waifu image as the main focus, with intuitive controls positioned for easy access without distraction. Image details will appear in an overlay or dedicated section upon click.
- Implement subtle fade-in animations for new images and category transitions, ensuring a smooth and delightful user experience. A gentle slide or fade animation for detail pop-ups.
